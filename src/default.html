PoC on automation with Azure

The purpose with this portal is to evaluate operation abilities for web based software.

After a commit and push in this repo on GitHub a Continuous Deployment will be made to https://cwpbeta-staging.azurewebsites.net

Following PowerShell code will publish site to https://cwpbeta.azurewebsites.net

Switch-AzureWebsiteSlot -Name cwpbeta -Force
